Frequently Asked Questions

F.A.Q.

Q: What is the purpose of this website?

+

A: To show some HTML, CSS, and JavaScript programming skills that might be usefull while creating other pages.

Q: Does this page stores any of my personal information?

+

A: NO. This page is intended only to test and display functionalities for HTML, JavaScript, and CSS, so no information is stored.

Feel free to play with the page and have fun (I will include some simple games).

Q: Does this page only uses HTML, CSS and JavaScript?

+

A: NO. Those are the main languages used, but some other libraries are also included like JQuery which is used for the unified navigation bar.

Q: Is this an original idea?

+

A: No, as shown in the Footer, these programs are based on Free Code Camp's post named: JavaScript Projects for Beginners

Nevertheless, I decided to put it all together to simulate a page with multiple sections and functionalities while dealing with a large file organization.

Q: Can I copy the code from this page?

+

A: Yes you can. If any of the functionalities of this page can be useful to you, then feel free to take it as a reference.

Do NOT forget that some (free) outsource resources could be included in this page, so don't forget to also copy the references or credits for the author(s).

Q: Is this the final version?

+

A: NO, I will slowly add more interesting programs as soon as I finish them.

If it gets too long, I might split it into different projects.
