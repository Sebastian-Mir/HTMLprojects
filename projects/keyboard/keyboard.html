How to play.


1. About the snake game

The game starts as soon as the page is loaded

Use the keyboard arrows to move the snake

Guide the snake to the Food object (green single square).

You can go across the walls


Have fun.


2. About the direction

When you press a direction key, the direction will change in the next move

You can observe the current direction with the emojis on the main screen

Example ⏩, ⏪, ⏫ or ⏬


Do your best.


3. About the score

Eating each snake food will give you 1 point

The snake will also grow in size by 1

The game ends when you eat yourself

when this happens, the score and the snake will

reset to 0 and the snake size of 5


Good luck.
